# Dusty Tab Notes — Chrome Web Store Listing

## Short Description (132 chars max)
A sidebar notepad for each Chrome tab group — with markdown editing, slash commands, and auto-save.

## Detailed Description

**Keep notes right where your work happens.**

Dusty Tab Notes gives you a sidebar notepad that's tied to your Chrome tab groups. Switch tab groups, and your notes switch with you. No more lost context jumping between projects.

**Tab Group Aware**
- Every tab group gets its own notepad — switch groups and the notes follow
- Ungrouped tabs get individual notepads
- Drag a tab into a group and its notes merge automatically

**Markdown Editing**
- Write in markdown with live preview
- Toggle between raw editing and rendered view
- Edit directly in the preview — it's fully interactive
- Cmd+Shift+S to strikethrough selected text

**Slash Commands**
Type `/` at the start of a line to insert:
- Headings (/h1, /h2, /h3)
- To-do items and checklists
- Code blocks
- Tables
- Dividers

**Interactive Checklists**
- Click checkboxes to toggle them
- Checked items automatically get strikethrough styling
- Great for quick task tracking alongside your research

**Auto-Save**
Everything saves automatically as you type. Your notes persist across browser sessions.

**Privacy First**
All notes are stored locally on your device using Chrome's built-in storage. Nothing is sent to any server. No accounts, no sign-ups, no tracking.
